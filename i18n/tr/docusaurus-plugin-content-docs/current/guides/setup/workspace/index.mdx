# Workspace Kurulumu

Workspace kurulumu en kapsamlı katılım sürecidir. Bir Workspace, CMS'nin tüm örneğine (ör. ana holding veya pazaryeri sahibi) sahip olan kök varlıktır.

Bu sihirbaz, altyapınızı hazır hale getirmek için tasarlanmış 6 temel aşamadan oluşur.
