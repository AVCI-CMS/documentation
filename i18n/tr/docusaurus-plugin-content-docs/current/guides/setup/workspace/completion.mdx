# 6. Tamamlama ve API Anahtarları

Tüm adımlar doğrulandığında, sihirbaz kurulumu tamamlar.

## Kritik Güvenlik Uyarısı

Son başarı ekranında, ana **Client ID** ve **API Key** bilgileriniz size sunulacaktır.

- **Hemen Kaydedin:** API Anahtarı **yalnızca bir kez** gösterilir. Kaybederseniz, Gelişmiş Ayarlar'dan yeni bir anahtar oluşturmanız gerekir, bu da mevcut frontend bağlantılarınızı bozar.
- **Sonraki Adımlar:** Bu kimlik bilgilerini Headless Next.js arayüzünüzün `.env` dosyasına kopyalayın.
