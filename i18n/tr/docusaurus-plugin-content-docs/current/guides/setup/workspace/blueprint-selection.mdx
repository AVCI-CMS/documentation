# 1. Şablon (Blueprint) Seçimi

Yeni bir Workspace kurmanın ilk adımı mimari **Şablonunuzu** seçmektir.

- **Plan Seçimi:** Standart SaaS planları veya Kurumsal paylaşımsız sunucular (dedicated clusters) arasında seçim yapın.
- **Şablon:** Temel kullanım durumunu (ör. "B2C E-ticaret", "B2B Toptan Satış", "Pazaryeri") seçin. Bu, yüzlerce dahili ayarı otomatik olarak yapılandırır.
- **Başlatma:** Seçildiğinde, sistem izole edilmiş veritabanı şemanızı arka planda hazırlayacaktır.
