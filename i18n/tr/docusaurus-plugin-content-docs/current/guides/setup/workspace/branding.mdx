# 3. Markalaşma (Branding)

Yeni CMS örneğinizin görsel kimliğini ayarlayın.

- **Workspace Adı:** Platformunuzun resmi adını girin (ör. "ACME Global").
- **Logolar:** Ana logonuzu ve favicon'unuzu yükleyin. Bunlar hem yönetici panelinde kullanılacak hem de e-postalara eklenecektir.
- **Ana Renkler:** Kurumsal kimliğinizle eşleşen renk (hex) kodlarını seçin.
