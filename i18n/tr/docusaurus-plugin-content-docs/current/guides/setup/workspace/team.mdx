# 5. Ekip Daveti

Platformu tek başınıza kurmak zorunda değilsiniz. 

- Kurucu ortaklarınızın veya baş geliştiricilerinizin e-posta adreslerini girin.
- Giriş yapabilmeleri ve kalan yapılandırmalara yardımcı olabilmeleri için onlara `Admin` yetkisi atayın.
