# 4. İletişim Ayarları

Platformun otomatik uyarıları (OTP'ler veya Sipariş Onayları gibi) nasıl göndereceğini belirleyin.

- **Varsayılan Yönlendirme:** AVCI CMS'nin yerleşik e-posta sunucularını kullanın (en hızlı yöntem).
- **Özel SMTP:** E-postaların doğrudan kendi alan adınızdan (domain) gelmesini sağlamak için SendGrid, Amazon SES veya Mailgun kimlik bilgilerinizi bağlayın.
