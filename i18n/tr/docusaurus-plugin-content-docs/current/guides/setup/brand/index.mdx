# Marka (Brand) Kurulumu

Bir **Marka** hesabı tipik olarak doğrudan tüketicilere satış yapmayan ancak ürün kataloglarını yöneten tedarikçiler veya resmi marka temsilcileri tarafından kullanılır.

Teknik adımlar Satıcı (Merchant) akışına benzese de, bağlam marka doğrulamasına odaklanmıştır.
