# Satıcı (Merchant) Kurulumu

Bir **Satıcı**, pazaryerinde faaliyet gösteren üçüncü taraf bir satıcıdır. Satıcılar fiziksel ürünlerle işlem yaptıkları ve ödeme (hakediş) aldıkları için kurulum akışları KYC (Müşterini Tanı) ve finansal düzenlemelere sıkı sıkıya bağlıdır.
