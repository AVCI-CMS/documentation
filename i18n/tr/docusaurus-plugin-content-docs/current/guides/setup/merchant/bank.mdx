# 3. Banka Bilgileri

Pazaryerinin ödemeleri (satıcıya aktarılan komisyonlar sonrası tutar) işleyebilmesi için gereklidir.

- **Hesap Sahibi:** Banka hesabındaki isim Şirket Unvanı ile (veya şahıs şirketleri için başvuru sahibiyle) eşleşmelidir. Üçüncü şahıs hesaplarına yapılan ödemeler varsayılan olarak engellenir.
- **IBAN Doğrulaması:** IBAN doğru ülke koduyla (ör. `TR`) başlamalı ve matematiksel checksum doğrulamalarından geçmelidir.
