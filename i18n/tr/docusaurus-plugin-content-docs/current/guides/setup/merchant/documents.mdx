# 4. Belgeler

Gerekli belge yüklemeleri, 2. Adımda seçilen Şirket Türüne göre dinamik olarak değişir.

### Standart Gereksinimler
- **Kimlik Belgesi:** Başvuru sahibinin fotoğraflı kimliği.
- **İmza Sirküleri:** Başvuru sahibinin şirket adına imza atmaya yetkili olduğunun kanıtı.
- **Pazaryeri Sözleşmesi:** Oluşturulan PDF'i indirmeli, imzalamalı ve ıslak imzalı kopyasını yüklemelisiniz.

### Özel Gereksinimler
- **Vergi Levhası:** Kullanıcı "Vergi Muafiyeti" kutusunu işaretlemedikçe zorunludur.
- **Ticaret Sicil Gazetesi:** Sadece Limited ve Anonim Şirketler için zorunludur.
