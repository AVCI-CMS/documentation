# 5. Beyanname

Son yasal onay adımı.

- Kullanıcı, yüklenen tüm bilgilerin doğru ve yasal olarak bağlayıcı olduğunu onaylayan kutuları açıkça işaretlemelidir.
- Bu adımı göndermek, satıcı durumunu `PENDING_APPROVAL` (bir Workspace Yöneticisinin incelemesi bekleniyor) veya `ACTIVE` (ayarlardan otomatik onay etkinleştirilmişse) olarak değiştirir.
