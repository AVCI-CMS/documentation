# Organizasyon (Organization) Kurulumu

Bir **Organizasyon** hesabı, B2B (İşletmeden İşletmeye) müşterileri için tasarlanmıştır. Bunlar toplu mal satın alan, kurumsal faturalandırma talep eden ve birden fazla alt çalışan hesabı yöneten kurumsal alıcılardır.
