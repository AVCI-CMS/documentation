# Webhook'lar

Webhook'lar, sisteminizde belirli bir olay (event) gerçekleştiğinde dış sistemlere (örneğin ERP sisteminize) anlık HTTP POST istekleri gönderilmesini sağlar.

## Desteklenen Olaylar (Events)
- `order.created`
- `order.status_changed`
- `product.created`
- `user.registered`
