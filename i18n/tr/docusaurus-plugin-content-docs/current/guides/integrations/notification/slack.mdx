# Slack Bildirimleri

Slack kanallarına anlık bildirimler gönderin

**Resmi Web Sitesi:** [https://slack.com](https://slack.com)

## Özellikler

- Webhook entegrasyonu
- Kanal bildirimleri
- Rich message format
- Anlık iletim

## Konfigürasyon Ayarları

Bu entegrasyonu aktif etmek için AVCI CMS yönetim panelinde aşağıdaki bilgileri sağlamanız gerekmektedir:

| Alan Adı | Etiket | Tür | Zorunlu | Açıklama |
|----------|--------|-----|---------|----------|
| `webhookUrl` | Webhook URL | `url` | ✅ Evet | Örn: https://hooks.slack.com/services/... |
| `channel` | Kanal | `text` | ✅ Evet | Örn: #genel |
| `username` | Bot Adı | `text` | ❌ Hayır | Örn: AVCI CMS Bot |

## Kurulum Adımları

### 1. Slack Uygulaması Oluşturun
Slack API panelinden yeni bir uygulama oluşturun ve bildirim göndermek istediğiniz workspace'e ekleyin.

### 2. Incoming Webhooks'u Aktif Edin
Uygulama ayarlarından 'Incoming Webhooks' özelliğini açın ve 'Add New Webhook to Workspace' butonuna tıklayarak kanalınızı seçin.

### 3. URL'yi Yapılandırın
Oluşturulan Webhook URL'sini kopyalayıp aşağıdaki alana yapıştırın.

## Geliştirici & API Entegrasyonu

Bu entegrasyon AVCI CMS arka planında `IntegrationType.SLACK` enum değeri ile yönetilmektedir. Yönetim paneli üzerinden yapılandırmayı tamamladıktan sonra sistem otomatik olarak `Slack` servislerini kullanmaya başlayacaktır.
