# Resend E-posta API

Geliştiriciler için modern ve güvenilir e-posta API servisi

**Resmi Web Sitesi:** [https://resend.com](https://resend.com)

## Özellikler

- Basit API key ile kullanım
- React Email desteği
- Domain doğrulama
- Webhook ve analytics

## Konfigürasyon Ayarları

Bu entegrasyonu aktif etmek için AVCI CMS yönetim panelinde aşağıdaki bilgileri sağlamanız gerekmektedir:

| Alan Adı | Etiket | Tür | Zorunlu | Açıklama |
|----------|--------|-----|---------|----------|
| `apiKey` | API Key | `password` | ✅ Evet | Örn: re_xxxxxxxx Resend API Key |
| `fromEmail` | Gönderen E-posta | `email` | ✅ Evet | Örn: onaylanmis@domain.com |
| `fromName` | Gönderen Adı | `text` | ✅ Evet | Örn: Şirket Adı |

## Geliştirici & API Entegrasyonu

Bu entegrasyon AVCI CMS arka planında `IntegrationType.RESEND` enum değeri ile yönetilmektedir. Yönetim paneli üzerinden yapılandırmayı tamamladıktan sonra sistem otomatik olarak `Resend` servislerini kullanmaya başlayacaktır.
