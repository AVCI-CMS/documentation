# HubSpot CRM

Müşteri ilişkileri yönetimi için HubSpot entegrasyonu

**Resmi Web Sitesi:** [https://www.hubspot.com](https://www.hubspot.com)

## Özellikler

- Contact yönetimi
- Lead tracking
- Deal pipeline
- Marketing automation

## Konfigürasyon Ayarları

Bu entegrasyonu aktif etmek için AVCI CMS yönetim panelinde aşağıdaki bilgileri sağlamanız gerekmektedir:

| Alan Adı | Etiket | Tür | Zorunlu | Açıklama |
|----------|--------|-----|---------|----------|
| `apiKey` | API Key | `password` | ✅ Evet | Örn: HubSpot API Key |
| `portalId` | Portal ID | `text` | ✅ Evet | Örn: HubSpot Portal ID |

## Geliştirici & API Entegrasyonu

Bu entegrasyon AVCI CMS arka planında `IntegrationType.HUBSPOT` enum değeri ile yönetilmektedir. Yönetim paneli üzerinden yapılandırmayı tamamladıktan sonra sistem otomatik olarak `HubSpot` servislerini kullanmaya başlayacaktır.
