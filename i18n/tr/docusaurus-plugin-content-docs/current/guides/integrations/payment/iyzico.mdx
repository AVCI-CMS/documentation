# Iyzico Ödeme Sistemi

Türkiye'nin önde gelen ödeme altyapısı ile güvenli ödeme

**Resmi Web Sitesi:** [https://www.iyzico.com](https://www.iyzico.com)

## Özellikler

- Kredi kartı ödemeleri
- Taksit seçenekleri
- 3D Secure
- Banka entegrasyonu

## Konfigürasyon Ayarları

Bu entegrasyonu aktif etmek için AVCI CMS yönetim panelinde aşağıdaki bilgileri sağlamanız gerekmektedir:

| Alan Adı | Etiket | Tür | Zorunlu | Açıklama |
|----------|--------|-----|---------|----------|
| `apiKey` | API Key | `text` | ✅ Evet | Örn: Iyzico API Key |
| `secretKey` | Secret Key | `password` | ✅ Evet | Örn: Iyzico Secret Key |
| `testMode` | Test Modu | `boolean` | ❌ Hayır | - |

## Geliştirici & API Entegrasyonu

Bu entegrasyon AVCI CMS arka planında `IntegrationType.IYZICO` enum değeri ile yönetilmektedir. Yönetim paneli üzerinden yapılandırmayı tamamladıktan sonra sistem otomatik olarak `Iyzico` servislerini kullanmaya başlayacaktır.
