# Google Analytics 4 (GA4)

AVCI CMS, GA4 E-ticaret etkinliklerini (view_item, add_to_cart, purchase) yerleşik olarak destekler.

- Kurulum için tek yapmanız gereken `G-XXXXXXXXXX` formatındaki **Measurement ID (Ölçüm Kimliği)** bilgisini panele girmektir.
- Sistem, Next.js vitrinine (storefront) gerekli kod parçacığını (snippet) otomatik olarak enjekte eder.
