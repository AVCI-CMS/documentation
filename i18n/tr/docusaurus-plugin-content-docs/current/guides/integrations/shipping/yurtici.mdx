# Yurtiçi Kargo 

Yurtiçi Kargo servisleri ile profesyonel lojistik yönetimi

**Resmi Web Sitesi:** [https://www.yurticikargo.com](https://www.yurticikargo.com)

## Özellikler

- Gelişmiş API desteği
- Self-servis kargo takibi
- Toplu gönderi operasyonları
- Özel raporlama

## Konfigürasyon Ayarları

Bu entegrasyonu aktif etmek için AVCI CMS yönetim panelinde aşağıdaki bilgileri sağlamanız gerekmektedir:

| Alan Adı | Etiket | Tür | Zorunlu | Açıklama |
|----------|--------|-----|---------|----------|
| `username` | Kullanıcı Adı | `text` | ✅ Evet | Örn: Yurtiçi Kargo Kullanıcı Adı |
| `password` | Şifre | `password` | ✅ Evet | Örn: Yurtiçi Kargo Şifresi |
| `testMode` | Test Modu | `boolean` | ❌ Hayır | - |

## Geliştirici & API Entegrasyonu

Bu entegrasyon AVCI CMS arka planında `IntegrationType.YURTICI` enum değeri ile yönetilmektedir. Yönetim paneli üzerinden yapılandırmayı tamamladıktan sonra sistem otomatik olarak `Yurtiçi Kargo` servislerini kullanmaya başlayacaktır.
