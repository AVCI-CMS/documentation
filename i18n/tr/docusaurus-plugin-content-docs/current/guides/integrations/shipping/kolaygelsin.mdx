# Kolay Gelsin Kargo 

Kolay Gelsin kargo servisleri entegrasyonu

**Resmi Web Sitesi:** [https://kolaygelsin.com](https://kolaygelsin.com)

## Özellikler

- Hızlı teslimat
- Anlık kargo takip
- Müşteri panelinden iptal/iade

## Konfigürasyon Ayarları

Bu entegrasyonu aktif etmek için AVCI CMS yönetim panelinde aşağıdaki bilgileri sağlamanız gerekmektedir:

| Alan Adı | Etiket | Tür | Zorunlu | Açıklama |
|----------|--------|-----|---------|----------|
| `musteri` | Müşteri Kodu | `text` | ✅ Evet | Örn: Kolay Gelsin Müşteri Kodu |
| `sifre` | Şifre | `password` | ✅ Evet | Örn: Kolay Gelsin Şifresi |
| `deliveryType` | Teslimat Tipi | `number` | ❌ Hayır | - |
| `testMode` | Test Modu | `boolean` | ❌ Hayır | - |

## Geliştirici & API Entegrasyonu

Bu entegrasyon AVCI CMS arka planında `IntegrationType.KOLAYGELSIN` enum değeri ile yönetilmektedir. Yönetim paneli üzerinden yapılandırmayı tamamladıktan sonra sistem otomatik olarak `Kolay Gelsin` servislerini kullanmaya başlayacaktır.
