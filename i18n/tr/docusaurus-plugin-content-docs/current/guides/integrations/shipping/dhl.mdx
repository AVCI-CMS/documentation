# DHL 

DHL Express global gönderim entegrasyonu

**Resmi Web Sitesi:** [https://www.dhl.com](https://www.dhl.com)

## Özellikler

- Yurtdışı kargo
- Hızlı teslimat
- Uluslararası ağ

## Konfigürasyon Ayarları

Bu entegrasyonu aktif etmek için AVCI CMS yönetim panelinde aşağıdaki bilgileri sağlamanız gerekmektedir:

| Alan Adı | Etiket | Tür | Zorunlu | Açıklama |
|----------|--------|-----|---------|----------|
| `siteId` | Site ID | `text` | ✅ Evet | Örn: DHL Site ID |
| `password` | Şifre | `password` | ✅ Evet | Örn: DHL Şifresi |
| `testMode` | Test Modu | `boolean` | ❌ Hayır | - |

## Geliştirici & API Entegrasyonu

Bu entegrasyon AVCI CMS arka planında `IntegrationType.DHL` enum değeri ile yönetilmektedir. Yönetim paneli üzerinden yapılandırmayı tamamladıktan sonra sistem otomatik olarak `DHL Express` servislerini kullanmaya başlayacaktır.
