# Netgsm (SMS)

Netgsm, Türkiye'de SMS gönderimleri için yaygın olarak kullanılan bir servis sağlayıcıdır.

- **Kullanıcı Adı / Şifre:** Netgsm API kimlik bilgileriniz.
- **Başlık (Header):** Operatör tarafından onaylanmış yasal gönderici başlığınız (örneğin `AVCI CMS`).

**Dikkat:** OTP'ler (Tek Kullanımlık Şifre) için iletim hızı daha yüksek olan Netgsm OTP API uç noktasını kullanmanız önerilir.
