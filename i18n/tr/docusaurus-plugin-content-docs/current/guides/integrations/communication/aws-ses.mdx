# AWS SES (E-posta)

Amazon Simple Email Service (SES), yüksek hacimli e-posta gönderimleri için uygun maliyetli bir çözümdür.

- SMTP kimlik bilgilerini sağlamanız veya IAM Rollarını kullanmanız gerekir.
- Alan adınızın DKIM ve SPF kayıtlarını AWS panelinden doğrulamayı unutmayın.
