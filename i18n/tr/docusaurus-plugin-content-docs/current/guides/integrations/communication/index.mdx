# İletişim Entegrasyonları

Kullanıcılarınıza OTP (Tek Kullanımlık Şifre), sipariş onayları ve kampanya duyuruları göndermek için SMS veya E-posta servis sağlayıcılarını sisteme bağlayabilirsiniz.
