# Slider Management

**Menu Path:** `Tools > Slider`

The **Slider** module manages the large, rotating banner images or videos usually found prominently on the homepage of your website. It is your primary tool for highlighting new collections, major sales, or key brand messages.

## Managing Your Homepage Carousel

A stale homepage can hurt conversions. You should frequently update your slider using this module.

### 1. Adding a New Slide
Click **"Create"** in the Slider menu.
- **Upload Media:** Select a high-resolution desktop image (e.g., 1920x1080) and a separate mobile-optimized image (e.g., 1080x1920). Providing both ensures the slide looks perfect on all devices without awkward cropping.
- **Title & Subtitle:** Write the text that will be overlaid on the image (e.g., "Summer Collection", "Up to 50% Off").
- **Call-to-Action (CTA):** Define the button text (e.g., "Shop Now") and the target URL (where the user goes when they click).

### 2. Ordering and Timing
- **Drag & Drop:** On the list page, grab the handle next to a slide and drag it up or down to change the rotation order.
- **Scheduling:** (Advanced) If supported by your theme, you can set a Start and End date for a slide. For example, you can create a "Black Friday" slide in October, set it to activate on November 24th, and auto-deactivate on November 27th.
