# NFC (Near Field Communication)

**Menu Path:** `Tools > NFC`

The **NFC** tool is a unique feature bridging your digital CMS with physical objects. It allows you to encode and manage NFC tags, which are small chips embedded in cards, stickers, or products that smartphones can read by tapping.

## Use Cases for NFC in E-Commerce
- **Smart Products:** Embed an NFC tag in a physical jacket. When the customer taps it, it opens a page proving authenticity and showing washing instructions.
- **Physical Events:** Create NFC wristbands for event check-in using the [Events Module](../modules/catalog/events/index.mdx) module.
- **Smart Business Cards:** Create a digital profile page and write it to a physical NFC card.

## Managing NFC Tags

1. **Registering a Tag:** Each physical NFC chip has a unique hardware ID (UID). You scan or enter this ID into the CMS to register it.
2. **Assigning an Action:** Click on the registered tag in the list.
   - Select what happens when a phone taps the tag.
   - *Link to URL:* E.g., `https://yourstore.com/product/123`
   - *Link to User Profile:* Opens the digital business card of a specific user.
3. **Analytics:** The system tracks how many times each specific NFC tag has been tapped, allowing you to measure engagement with your physical products.
