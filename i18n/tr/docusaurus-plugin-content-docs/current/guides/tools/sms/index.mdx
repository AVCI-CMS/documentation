# SMS Tool

**Menu Path:** `Tools > SMS`

The **SMS Tool** functions very similarly to the Mail Tool, but is strictly for text messages. Because SMS has a near 98% open rate, it is highly effective for critical, time-sensitive updates.

## Best Practices for SMS
- **Keep it Short:** SMS messages are billed per segment (usually 160 characters). If you write a long message, you will be billed for 2 or 3 SMS messages per customer.
- **Urgency:** Use SMS for One-Time Passwords (OTP), delivery today alerts, or extremely limited flash sales. Do not use SMS for long newsletters.

## Monitoring SMS Delivery
Just like emails, you can check the **Logs** tab.
- If an SMS fails, it is usually because the customer entered a landline number or an invalid country code during checkout.
- You can manually resend an SMS or compose a custom message to a specific phone number directly from this interface.
