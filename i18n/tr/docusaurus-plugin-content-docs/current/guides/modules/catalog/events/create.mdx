# Create Event

**Menu Path:** `Catalog > Events > Create`

Creating an event requires specific temporal and spatial data unlike standard retail products. The creation form captures the absolute basics to get the event into the system.

## Step 1: Core Details
- **Event Title:** Make it catchy and descriptive.
- **Event Type:** Select between:
  - *Physical:* Requires a geographical location.
  - *Online / Webinar:* Requires a digital meeting link (Zoom, Google Meet, etc.).
- **Description:** Provide a rich-text overview. What will attendees learn? Who is performing? Include schedules or formatting as needed.

## Step 2: Time & Scheduling
Accurate time data is critical for ticketing and calendar integrations.
- **Start Date & Time:** When does the event officially begin?
- **End Date & Time:** When does it conclude?
- **Registration Deadline:** (Optional) If you need to finalize catering or seating, set a cutoff date. After this date, tickets can no longer be purchased even if the event hasn't started.

## Step 3: Location (Physical Events)
If you selected "Physical" as the event type, you must define where it happens.
- **Select a Place:** The system connects to the `Relations > Places` module. Search for a pre-defined venue (e.g., "Convention Center A"). This automatically pulls in the address, map coordinates, and maximum venue capacity.

## Step 4: Media
- **Cover Image / Banner:** Upload a high-resolution 16:9 banner. This is heavily used in frontend listings and social media sharing (OpenGraph).

> Click **"Save"** to generate the event. To configure tickets, add speakers, or build a detailed agenda, proceed to the [Edit Event](./edit.mdx) screen.
