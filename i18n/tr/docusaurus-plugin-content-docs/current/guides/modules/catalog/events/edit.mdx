# Edit Event & Ticketing

**Menu Path:** `Catalog > Events > Edit`

The Event Edit screen is where the logistics and monetization of your event are configured. It contains several advanced tabs.

## 1. Ticketing & Capacities Tab
This is the most critical section for monetized events. You can create multiple tiers of tickets.

### Creating a Ticket Tier
Click **"Add Ticket Type"** and define:
- **Ticket Name:** e.g., "General Admission", "VIP Backstage", "Early Bird".
- **Price:** Set the cost. Enter 0 for free events.
- **Quota (Capacity):** Limit the number of tickets available for this specific tier (e.g., only 50 VIP tickets).
- **Sales Window:** Define precisely when this ticket goes on sale and when it goes off sale. 
  - *Example:* "Early Bird" is available from Jan 1 to Jan 15. "General Admission" opens on Jan 16.

## 2. Agenda & Schedule Tab
For multi-day conferences or structured seminars, build a visual schedule.
- **Add Session:** Define a sub-event (e.g., "Opening Keynote", "Lunch Break").
- **Time Slot:** Assign start and end times for the session.
- **Description:** Briefly explain the session topic.

## 3. Speakers & Guests Tab
Highlight who is presenting at the event.
- **Assign Speakers:** Link existing users (who have a Speaker profile) to the event. Their profile pictures, names, and biographies will automatically populate on the frontend event page.

## 4. Attendees Management
Once the event is live and selling, this tab populates with real data.
- **Guest List:** View everyone who has purchased a ticket or RSVP'd.
- **Export:** Download the attendee list as a CSV or Excel file for use at the check-in desk.
- **Manual Addition:** Administrators can manually add guests (e.g., sponsors or press) bypassing the payment gateway.

> **Going Live:** Ensure your ticket sales windows are correct and the Event Status is set to **"Active"** so users can begin registering.
