# Coupons

**Menu Path:** `Catalog > Coupons`

While Campaigns apply automatically to eligible products, **Coupons** require the customer to actively enter a specific alphanumeric code at the checkout stage. They are essential for influencer marketing, targeted email campaigns, and customer retention strategies.

## Deep Dive into Coupon Creation

Navigate to **Catalog > Coupons** and click **"Create"**.

### 1. Code Generation
- **Coupon Code:** You can type a custom code (e.g., `WELCOME20`) or click the "Generate Random" button for unique tracking codes.

### 2. Discount Configuration
- **Discount Type:** Choose between:
  - *Percentage (%):* Reduces the cart subtotal by a percentage.
  - *Fixed Amount ($):* A flat monetary deduction.
  - *Free Shipping:* Eliminates shipping costs regardless of the order size.
- **Amount:** Enter the specific value of the discount.

### 3. Strict Usage Limits
Protect your margins by applying limits:
- **Minimum Spend Requirement:** The coupon is only valid if the cart totals at least $50.
- **Total Usage Limit:** The coupon will expire globally after it has been used X times (e.g., "First 100 customers only!").
- **Per-User Limit:** Restrict usage so a single logged-in account can only use the code once.

### 4. Applicability Rules
Just like Campaigns, you can restrict coupons:
- Exclude specific products that are already heavily discounted.
- Limit the coupon to only apply to a specific Category or Brand.

> **Tracking & Analytics:** Once a coupon is live, returning to the Edit screen will show you exactly how many times it has been used and the total revenue generated through that specific code.
