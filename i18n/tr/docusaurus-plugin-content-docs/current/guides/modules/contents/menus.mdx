# Menus

**Menu Path:** `Contents > Menus`

The **Menus** module is where you configure the navigation areas of your site such as the top navigation bar (Header Nav), footer, mobile menu using the drag-and-drop method.

You design how users will navigate your site through this module.

## Creating a New Menu (Menu Group)

Generally, your site has predefined menu areas ("Main Menu", "Footer Links"). To add a new menu group:

1. Go to the **Contents > Menus** screen.
2. Click the **"Create Menu Group"** option.
3. Give the menu a name (e.g., "Corporate Footer") and a system identifier (Identifier).

## Editing Menu Content (Links)

To add a link to an existing menu or change its order:

1. Click on the menu group you want to edit.
2. Press the **"Add Link"** button on the opened menu tree (Tree) interface.
3. Select the properties of the link you will add:
   - **Label:** The text the user will see (e.g., "About Us").
   - **Link Type:** 
      - *Page:* You can directly select a Page (Pages) you created in the system.
      - *Category:* You can redirect to a product or blog category.
      - *Custom URL:* You can enter a completely different external link (`https://...`).
4. **Drag & Drop:** You can change the order of menu items by dragging them with your mouse or create a **Submenu (Dropdown)** by sliding an item to the right below another.
5. Save the changes to update your site's navigation.
