# Common (Ortak Modüller)

**Common** grubu, AVCI CMS içerisinde tek başına kök bir menü öğesi olarak bulunmayan ancak **Katalog (Ürünler)**, **İçerik (Blog)** veya **İlişkiler (Satıcılar)** gibi birçok farklı modülün altında sekme veya alt modül olarak çalışan yapıtaşlarını barındırır.

Örneğin, "Kategoriler" veya "Yorumlar" mantığı ürünlerde, blog yazılarında veya haberlerde tamamen aynı altyapı ve arayüz ile çalışır. Bu rehberde ortak modüllerin nasıl yönetildiğini öğrenebilirsiniz.
