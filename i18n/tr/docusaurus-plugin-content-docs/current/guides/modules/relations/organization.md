# Organization Frontend Usage Guide

This guide covers the Organization module in the AVCI CMS frontend.

## Overview
Organizations represent physical venues, institutions, or corporate entities. They can have locations, capacities, and host events.

## Key Features
- **Organization Settings**: Track locations, capacity, and status.
- **Roles & Permissions**: Fine-grained access control using `PermissionService`.
- **Deletion Workflows**: Soft deletion followed by permanent deletion.

## Common Operations
- **Listing Organizations**: Available at `/organizations`.
- **Management**: Update address, maps (latitude/longitude), and team assignments.
