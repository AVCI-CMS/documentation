# Çalışma Alanı Kurulumuna Genel Bakış

Çalışma Alanı (Workspace) Kurulum sihirbazına hoş geldiniz.

Çalışma alanı kurulumu, platformun en temel yapı taşıdır. Kendi pazar yerinizi (marketplace) veya B2B toptan satış platformunuzu başlatırken altyapıyı şekillendirdiğiniz ilk yer burasıdır.

## Kurulum Adımları

Yeni bir çalışma alanı oluştururken aşağıdaki adımları tamamlarsınız:

1. **Markalaşma (Branding):** Platformun dışa dönük adı, kurumsal renkleri ve logoları.
2. **Fatura (Billing):** Platform kullanım (SaaS) aboneliğiniz için fatura bilgileriniz.
3. **İletişim (Communication):** Sistem e-postalarının hangi e-posta sağlayıcısı üzerinden çıkacağı.
4. **Altyapı Seçimi (Blueprint):** İş modelinize uygun mimarinin (B2C, B2B, Marketplace) seçilmesi.
5. **Ekip (Team):** Diğer kurucu veya yöneticilerin çalışma alanına davet edilmesi.
6. **Tamamlama (Completion):** Sistem derlemesinin (build) başlaması.
