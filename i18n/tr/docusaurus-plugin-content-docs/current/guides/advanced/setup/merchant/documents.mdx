# 3. Belge Yükleme

**Rota:** `(setup) > Setup > Merchant > Step 3`

Satıcıların platformda güvenle ticaret yapabilmesi için yasal şirket belgelerini (KYC) sisteme yüklemeleri zorunludur.

## Gerekli Yüklemeler

Platformun yasal gereksinimlerine göre genellikle şu belgeler istenir:
- **Vergi Levhası:** Güncel yıla ait onaylı vergi levhası.
- **İmza Sirküleri veya Beyannamesi:** Yetkili kişiye ait imzayı doğrulayan resmi belge.
- **Kimlik Fotokopisi:** Şahıs şirketi ise firma sahibinin kimlik görüntüsü.

> [!WARNING]
> Okunaksız, bulanık veya tarihi geçmiş belgeler yüklemek mağaza onay sürecinizin uzamasına veya reddedilmesine yol açacaktır. Lütfen pdf veya yüksek çözünürlüklü jpg/png dosyaları yükleyin.
