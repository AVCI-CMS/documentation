# 4. Banka ve Hakediş Detayları

**Rota:** `(setup) > Setup > Merchant > Step 4`

Satışlardan elde edeceğiniz hakedişlerin (payouts) aktarılacağı ticari banka hesabınızı tanımladığınız bölümdür.

## Gerekli Bilgiler

- **Hesap Sahibi Adı:** Adım 2'de girdiğiniz resmi şirket unvanı ile **birebir aynı** olmalıdır. Başkasına ait (şahsi) hesaplar kabul edilmez.
- **IBAN:** 26 haneli, TR ile başlayan banka hesap numaranız.

:::info Hakediş Süreci
Platformun hakediş periyodu ayarlarına (Haftalık, Aylık vb.) göre bakiyeniz otomatik olarak bu IBAN'a aktarılacaktır.
:::
