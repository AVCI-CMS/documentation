# Satıcı Profili

**Rota:** `(merchant) > /merchant/profile`

Profil bölümü, satıcıların mağazalarının müşterilere vitrinde (storefront) nasıl görüneceğini yönetmelerine olanak tanır.

## Temel Özellikler

- **Mağaza Adı:** Mağazanızın müşterilere dönük genel adı.
- **Mağaza Açıklaması:** İşletme geçmişinizi, iade politikalarınızı veya uzmanlık alanlarınızı açıklayabileceğiniz zengin metin (rich-text) alanı.
- **Görsel Varlıklar:** Mağazanızın Logosunu (1:1 oranı önerilir) ve Kapak Afişini (16:9 oranı önerilir) yükleyin veya güncelleyin.
- **İletişim Bilgileri:** Herkese açık olarak gösterilecek müşteri destek e-postası ve telefon numarası.

:::info Önbellek (Cache) Geçersiz Kılma
Profil görsellerinizde (logo gibi) yapılan değişikliklerin İçerik Dağıtım Ağı (CDN) önbelleklemesi nedeniyle canlı vitrine yansıması birkaç dakika sürebilir.
:::
