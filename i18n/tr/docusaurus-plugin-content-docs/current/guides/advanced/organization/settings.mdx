# Kurumsal Ayarlar

**Rota:** `(organization) > /organization/settings`

Bağlı varlıklara (alt satıcılara) doğru aşağı yönlü kademeli olarak etki eden üst düzey operasyonel kuralları yapılandırın.

## Yapılandırma Alanları

- **Alt Satıcı Oluşturma (Sub-Merchant Creation):** Çalışma Alanı Yöneticisi (Workspace Admin) tarafından etkinleştirildiyse, bölge yöneticilerinizin bağımsız olarak yeni alt satıcılar oluşturup oluşturamayacağını veya Organizasyon Yöneticisinin onayına ihtiyaç duyup duymayacaklarını yapılandırabilirsiniz.
- **Veri Siloları (Multi-tenant izolasyonu):** Organizasyon düzeyinde atanan personelin *tüm* satıcılardan gelen toplu verileri görüp göremeyeceğini veya belirli satıcılara açıkça (manuel olarak) atanmaları gerekip gerekmediğini ayarlayın.
- **API Erişimi:** Platformu konsolide (toplu) satış verilerini çekecek kurumsal ERP'nize (SAP veya Oracle gibi) bağlamak için Kurumsal API anahtarları (Enterprise API Keys) oluşturun.
