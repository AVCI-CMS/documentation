# Rich Text Editor

When writing Product Descriptions, Blog Posts, or Custom Pages, AVCI CMS provides a WYSIWYG (What You See Is What You Get) Rich Text Editor. You do not need to know HTML to create beautiful layouts.

## Standard Formatting Tools

The toolbar at the top of the editor works very similarly to Microsoft Word or Google Docs.
- **Typography:** Bold, Italic, Underline, and Strikethrough.
- **Headings:** Use the Heading dropdown to insert `H2`, `H3`, and `H4` tags. *SEO Tip: Never use an `H1` tag inside the editor, as the system automatically uses the Page Title or Product Name as the `H1`.*
- **Lists:** Bulleted and numbered lists for easy-to-read feature breakdowns.
- **Alignment:** Left, Center, Right, or Justify text.

## Advanced Insertions

### 1. Adding Images
- Click the **Image Icon** in the toolbar.
- You can either paste an external image URL, or click **"Browse Media"** to open the [Media Library](../tools/media.mdx). 
- Select an existing image or upload a new one directly from your computer. Once inserted, you can drag the corners of the image to resize it within the text.

### 2. Embedding Videos
- Click the **Video/Media Icon**.
- Paste a YouTube or Vimeo link. The editor will automatically convert the link into a responsive video player that works perfectly on both desktop and mobile devices.

### 3. Hyperlinks
- Highlight the text you want to make clickable.
- Click the **Link (Chain) Icon**.
- Enter the destination URL. If you are linking to an external site (not your own store), it is best practice to check the "Open in new tab" option so you don't navigate the user away from your checkout flow.

### 4. Tables
- Click the **Table Icon** to insert a grid.
- Tables are excellent for displaying technical specifications (e.g., Size, Weight, Material) on product pages. You can right-click inside a table to add or remove rows and columns.
