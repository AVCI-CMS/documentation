# Dashboard

**Menu Path:** `Dashboard`

The **Dashboard** is the first comprehensive summary screen you encounter when logging into AVCI CMS. Through this screen, you can instantly access important metrics, quick shortcuts, and recent activities belonging to the workspace you manage.

## What Can You Do?

The dashboard provides a bird's-eye view of all modules in the system:

1. **Summary Metrics:** You can graphically review the summary of order numbers, new registered users, current visitor statistics, and financial movements.
2. **Quick Actions:** You can use shortcuts that provide direct access to frequently performed operations, such as adding a new product, creating a quick campaign, or approving recent comments.
3. **Notification Center:** You can see important notifications, critical stock alerts, or system messages related to your workspace on this page.

## How to Use It?

- **Date Filtering:** By using the date filter in the top right corner of the page (e.g., "Last 7 Days", "This Month"), you can update all statistics on the screen according to the time range you wish.
- **Clicking Module Cards:** When you click on any summary card on the Dashboard (for example, "Pending Orders"), the system automatically redirects you to the detail page of the relevant module (Orders).

> **Tip:** The Dashboard screen is specific to managers. If you cannot see some metrics or charts you are looking for, your account's permission settings might be restricted. You can consult your system administrator for a permission request via **Workspace Settings > Authority Settings**.
