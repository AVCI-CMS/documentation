# 4. Communication Settings

Determine how the platform sends automated alerts (like OTPs or Order Confirmations).

- **Default Routing:** Use AVCI CMS's built-in email servers (quickest method).
- **Custom SMTP:** Connect your own SendGrid, Amazon SES, or Mailgun credentials to ensure emails come directly from your domain.
