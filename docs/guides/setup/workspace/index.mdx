# Workspace Setup

The Workspace setup is the most comprehensive onboarding flow. A Workspace is the root entity that owns the entire instance of the CMS (e.g., the main holding company or the marketplace owner).

This wizard consists of 6 core stages designed to get your infrastructure ready.
