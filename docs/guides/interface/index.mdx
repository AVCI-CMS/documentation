# User Interface & UX Guide

Welcome to the AVCI CMS! Whether you are a store manager, an accountant, or a customer support agent, the platform is designed to be intuitive and highly efficient. 

Instead of re-learning how to use every single module, AVCI CMS uses a **Consistent Design System**. Once you learn how to filter a data table or save a form in the "Products" module, you will instantly know how to do it in the "Users" or "Orders" modules.

## Getting Started
Before diving into specific business operations, we highly recommend reading through this interface guide to familiarize yourself with the core UI components.
