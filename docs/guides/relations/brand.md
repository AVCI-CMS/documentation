# Brand Frontend Usage Guide

This guide explains how to use the Brand module within the AVCI CMS frontend.

## Overview
Brands are typically groups that have multiple products or events and operate at a higher level than individual merchants. 

## Key Features
- **Brand Management**: Centralized hub to manage brand details, assets, and descriptions.
- **Team Management**: Manage who has access to the brand's settings.
- **Documents & Verification**: Upload and verify brand documents.

## Common Operations
- **Listing Brands**: View the brand list via `/brands`.
- **Status Workflows**: Brands can move through statuses like `PENDING_VALIDATION`, `ACTIVE`, `SUSPENDED`, and `DELETED`.
