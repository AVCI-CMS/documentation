# Merchant Frontend Usage Guide

This guide explains how to use the Merchant module within the AVCI CMS frontend.

## Overview
Merchants represent individual sellers, stores, or retail locations within the system. They can list products, manage team members, and configure settings such as payments, shipping, and holiday mode.

## Key Features
- **Merchant Management**: Create, update, suspend, or delete merchants.
- **Documents**: Upload and verify merchant legal documents.
- **Team Management**: Add team members and manage their roles/permissions.
- **Settings**: Manage bank accounts, financial settings, and more.

## Common Operations
- **Listing Merchants**: Navigate to `/merchants` to view the paginated list of all merchants.
- **Creating a Merchant**: Use the "Create" button on the listing page to add a new merchant.
- **Deleting a Merchant**: For merchants with `DELETED` status, a permanent delete option will be available which cascade deletes related products and images.
