# Iyzico Payment System

Secure payment with Turkey's leading payment infrastructure

**Official Website:** [https://www.iyzico.com](https://www.iyzico.com)

## Features

- Credit card payments
- Installment options
- 3D Secure
- Bank integration

## Configuration Settings

To activate this integration, you need to provide the following information in the AVCI CMS admin panel:

| Field Name | Label | Type | Required | Description |
|------------|-------|------|----------|-------------|
| `apiKey` | API Key | `text` | ✅ Yes | Ex: Iyzico API Key |
| `secretKey` | Secret Key | `password` | ✅ Yes | Ex: Iyzico Secret Key |
| `testMode` | Test Mode | `boolean` | ❌ No | - |

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.IYZICO` enum value. After completing the configuration via the admin panel, the system will automatically start using `Iyzico` services.
