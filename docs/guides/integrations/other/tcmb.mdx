# TCMB Döviz Kurları

Türkiye Cumhuriyeti Merkez Bankası üzerinden günlük döviz kurlarını çekin

**Resmi Web Sitesi:** [https://www.tcmb.gov.tr](https://www.tcmb.gov.tr)

## Özellikler

- Resmi günlük döviz kurları
- Çapraz kur hesaplama desteği
- Otomatik güncelleme

## Konfigürasyon Ayarları

Bu entegrasyonu aktif etmek için AVCI CMS yönetim panelinde aşağıdaki bilgileri sağlamanız gerekmektedir:

| Alan Adı | Etiket | Tür | Zorunlu | Açıklama |
|----------|--------|-----|---------|----------|
| `autoUpdate` | Otomatik Güncelleme | `boolean` | ❌ Hayır | - |

## Geliştirici & API Entegrasyonu

Bu entegrasyon AVCI CMS arka planında `IntegrationType.TCMB` enum değeri ile yönetilmektedir. Yönetim paneli üzerinden yapılandırmayı tamamladıktan sonra sistem otomatik olarak `TCMB` servislerini kullanmaya başlayacaktır.
