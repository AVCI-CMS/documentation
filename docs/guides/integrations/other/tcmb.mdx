# CBRT Exchange Rates

Fetch daily exchange rates from the Central Bank of the Republic of Turkey

**Official Website:** [https://www.tcmb.gov.tr](https://www.tcmb.gov.tr)

## Features

- Official daily exchange rates
- Cross rate calculation support
- Automatic update

## Configuration Settings

To activate this integration, you need to provide the following information in the AVCI CMS admin panel:

| Field Name | Label | Type | Required | Description |
|------------|-------|------|----------|-------------|
| `autoUpdate` | Automatic Update | `boolean` | ❌ No | - |

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.TCMB` enum value. After completing the configuration via the admin panel, the system will automatically start using `TCMB` services.
