# ClickUp Project Management

Task tracking with modern project management platform

**Official Website:** [https://clickup.com](https://clickup.com)

## Features

- Task management
- Project tracking
- Time management
- Team collaboration

## Configuration Settings

To activate this integration, you need to provide the following information in the AVCI CMS admin panel:

| Field Name | Label | Type | Required | Description |
|------------|-------|------|----------|-------------|
| `apiKey` | API Key | `password` | ✅ Yes | Ex: ClickUp API Key |
| `teamId` | Team ID | `text` | ✅ Yes | Ex: ClickUp Team ID |
| `spaceId` | Space ID | `text` | ❌ No | Ex: ClickUp Space ID |
| `listId` | List ID | `text` | ❌ No | Ex: ClickUp List ID |

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.CLICKUP` enum value. After completing the configuration via the admin panel, the system will automatically start using `ClickUp` services.
