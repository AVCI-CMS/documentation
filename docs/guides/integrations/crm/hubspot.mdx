# HubSpot CRM

HubSpot integration for customer relationship management

**Official Website:** [https://www.hubspot.com](https://www.hubspot.com)

## Features

- Contact management
- Lead tracking
- Deal pipeline
- Marketing automation

## Configuration Settings

To activate this integration, you need to provide the following information in the AVCI CMS admin panel:

| Field Name | Label | Type | Required | Description |
|------------|-------|------|----------|-------------|
| `apiKey` | API Key | `password` | ✅ Yes | Ex: HubSpot API Key |
| `portalId` | Portal ID | `text` | ✅ Yes | Ex: HubSpot Portal ID |

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.HUBSPOT` enum value. After completing the configuration via the admin panel, the system will automatically start using `HubSpot` services.
