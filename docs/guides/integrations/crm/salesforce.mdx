# Salesforce CRM

Customer management with the world's leading CRM platform

**Official Website:** [https://www.salesforce.com](https://www.salesforce.com)

## Features

- Lead management
- Opportunity tracking
- Account management
- Sales automation

## Configuration Settings

To activate this integration, you need to provide the following information in the AVCI CMS admin panel:

| Field Name | Label | Type | Required | Description |
|------------|-------|------|----------|-------------|
| `clientId` | Client ID | `text` | ✅ Yes | Ex: Salesforce Client ID |
| `clientSecret` | Client Secret | `password` | ✅ Yes | Ex: Salesforce Client Secret |
| `username` | Username | `text` | ✅ Yes | Ex: Salesforce username |
| `password` | Password | `password` | ✅ Yes | Ex: Salesforce password |
| `securityToken` | Security Token | `password` | ✅ Yes | Ex: Salesforce Security Token |

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.SALESFORCE` enum value. After completing the configuration via the admin panel, the system will automatically start using `Salesforce` services.
