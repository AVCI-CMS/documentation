# Mailjet Email Service

Manage bulk email campaigns with a reliable email delivery service

**Official Website:** [https://www.mailjet.com](https://www.mailjet.com)

## Features

- Bulk email sending
- Template management
- Analytics and reporting
- SMTP and API support

## Configuration Settings

To activate this integration, you need to provide the following information in the AVCI CMS admin panel:

| Field Name | Label | Type | Required | Description |
|------------|-------|------|----------|-------------|
| `apiKey` | API Key | `text` | ✅ Yes | Ex: Enter your Mailjet API Key |
| `apiSecret` | API Secret | `password` | ✅ Yes | Ex: Enter your Mailjet API Secret |
| `fromEmail` | From Email | `email` | ✅ Yes | Ex: example@domain.com |
| `fromName` | From Name | `text` | ✅ Yes | Ex: Company Name |

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.MAILJET` enum value. After completing the configuration via the admin panel, the system will automatically start using `Mailjet` services.
