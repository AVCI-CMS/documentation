# SendGrid E-posta Platformu

Yüksek performanslı e-posta gönderim platformu ile ulaşılabilirliği artırın

**Resmi Web Sitesi:** [https://sendgrid.com](https://sendgrid.com)

## Özellikler

- Yüksek ulaşılabilirlik
- Gelişmiş analytics
- Template engine
- Webhook desteği

## Konfigürasyon Ayarları

Bu entegrasyonu aktif etmek için AVCI CMS yönetim panelinde aşağıdaki bilgileri sağlamanız gerekmektedir:

| Alan Adı | Etiket | Tür | Zorunlu | Açıklama |
|----------|--------|-----|---------|----------|
| `apiKey` | API Key | `password` | ✅ Evet | Örn: SendGrid API Key'inizi girin |
| `fromEmail` | Gönderen E-posta | `email` | ✅ Evet | Örn: ornek@domain.com |
| `fromName` | Gönderen Adı | `text` | ✅ Evet | Örn: Şirket Adı |

## Geliştirici & API Entegrasyonu

Bu entegrasyon AVCI CMS arka planında `IntegrationType.SENDGRID` enum değeri ile yönetilmektedir. Yönetim paneli üzerinden yapılandırmayı tamamladıktan sonra sistem otomatik olarak `SendGrid` servislerini kullanmaya başlayacaktır.
