# Netgsm Integration

Netgsm is a popular local SMS provider in Turkey, known for its cost-effectiveness and localized support.

## Setup
1. Obtain your **Username**, **Password**, and official **Sender ID** (Başlık) from the Netgsm panel.
2. In AVCI CMS, go to **Workspace > Settings > Communication > SMS**.
3. Select **Netgsm**.
4. Input your credentials. Note that your Sender ID must exactly match the one approved by Netgsm (case-sensitive).
