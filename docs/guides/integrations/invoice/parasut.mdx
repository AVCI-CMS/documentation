# Paraşüt E-Fatura 

Paraşüt API üzerinden otomatik e-fatura, e-arşiv ve müşteri yönetimi

**Resmi Web Sitesi:** [https://www.parasut.com](https://www.parasut.com)

## Özellikler

- Otomatik E-Fatura oluşturma
- E-Arşiv desteği
- Müşteri otomatik senkronizasyonu
- Ödeme takibi

## Konfigürasyon Ayarları

Bu entegrasyonu aktif etmek için AVCI CMS yönetim panelinde aşağıdaki bilgileri sağlamanız gerekmektedir:

| Alan Adı | Etiket | Tür | Zorunlu | Açıklama |
|----------|--------|-----|---------|----------|
| `companyId` | Firma ID | `text` | ✅ Evet | Örn: Paraşüt Firma ID |
| `clientId` | Client ID (Uygulama ID) | `text` | ✅ Evet | Örn: Paraşüt Uygulama Client ID |
| `clientSecret` | Client Secret (Uygulama Secret) | `password` | ✅ Evet | Örn: Paraşüt Uygulama Client Secret |
| `username` | Kullanıcı Adı (Email) | `email` | ✅ Evet | Örn: Paraşüt giriş e-postası |
| `password` | Şifre | `password` | ✅ Evet | Örn: Paraşüt şifreniz |
| `testMode` | Sandbox Modu | `boolean` | ❌ Hayır | - |

## Geliştirici & API Entegrasyonu

Bu entegrasyon AVCI CMS arka planında `IntegrationType.PARASUT` enum değeri ile yönetilmektedir. Yönetim paneli üzerinden yapılandırmayı tamamladıktan sonra sistem otomatik olarak `Paraşüt` servislerini kullanmaya başlayacaktır.
