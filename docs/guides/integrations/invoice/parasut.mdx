# Paraşüt E-Invoice

Automatic e-invoice, e-archive and customer management via Paraşüt API

**Official Website:** [https://www.parasut.com](https://www.parasut.com)

## Features

- Automatic E-Invoice generation
- E-Archive support
- Automatic customer synchronization
- Payment tracking

## Configuration Settings

To activate this integration, you need to provide the following information in the AVCI CMS admin panel:

| Field Name | Label | Type | Required | Description |
|------------|-------|------|----------|-------------|
| `companyId` | Company ID | `text` | ✅ Yes | Ex: Paraşüt Company ID |
| `clientId` | Client ID (App ID) | `text` | ✅ Yes | Ex: Paraşüt App Client ID |
| `clientSecret` | Client Secret (App Secret) | `password` | ✅ Yes | Ex: Paraşüt App Client Secret |
| `username` | Username (Email) | `email` | ✅ Yes | Ex: Paraşüt login email |
| `password` | Password | `password` | ✅ Yes | Ex: Your Paraşüt password |
| `testMode` | Sandbox Mode | `boolean` | ❌ No | - |

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.PARASUT` enum value. After completing the configuration via the admin panel, the system will automatically start using `Paraşüt` services.
