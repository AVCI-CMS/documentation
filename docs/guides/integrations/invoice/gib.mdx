# GİB E-Arşiv Portal

Gelir İdaresi Başkanlığı E-Arşiv portalı üzerinden ücretsiz fatura kesimi

**Resmi Web Sitesi:** [https://earsivportal.efatura.gov.tr](https://earsivportal.efatura.gov.tr)

## Özellikler

- Ücretsiz E-Arşiv fatura
- Resmi portal entegrasyonu
- Fatura sorgulama

## Konfigürasyon Ayarları

Bu entegrasyonu aktif etmek için AVCI CMS yönetim panelinde aşağıdaki bilgileri sağlamanız gerekmektedir:

| Alan Adı | Etiket | Tür | Zorunlu | Açıklama |
|----------|--------|-----|---------|----------|
| `username` | Kullanıcı Kodu / VKN | `text` | ✅ Evet | Örn: Portal kullanıcı kodu |
| `password` | Şifre | `password` | ✅ Evet | Örn: Portal şifresi |
| `testMode` | Test Portalı | `boolean` | ❌ Hayır | - |

## Geliştirici & API Entegrasyonu

Bu entegrasyon AVCI CMS arka planında `IntegrationType.GIB` enum değeri ile yönetilmektedir. Yönetim paneli üzerinden yapılandırmayı tamamladıktan sonra sistem otomatik olarak `GİB Portal` servislerini kullanmaya başlayacaktır.
