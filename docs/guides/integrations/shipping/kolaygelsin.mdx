# Kolay Gelsin Kargo

Kolay Gelsin cargo services integration

**Official Website:** [https://kolaygelsin.com](https://kolaygelsin.com)

## Features

- Fast delivery
- Real-time cargo tracking
- Cancellation/return from customer panel

## Configuration Settings

To activate this integration, you need to provide the following information in the AVCI CMS admin panel:

| Field Name | Label | Type | Required | Description |
|------------|-------|------|----------|-------------|
| `musteri` | Customer Code | `text` | ✅ Yes | Ex: Kolay Gelsin Customer Code |
| `sifre` | Password | `password` | ✅ Yes | Ex: Kolay Gelsin Password |
| `deliveryType` | Delivery Type | `number` | ❌ No | - |
| `testMode` | Test Mode | `boolean` | ❌ No | - |

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.KOLAYGELSIN` enum value. After completing the configuration via the admin panel, the system will automatically start using `Kolay Gelsin` services.
