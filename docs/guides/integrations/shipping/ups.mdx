# UPS Kargo

Global delivery solutions with UPS Kargo

**Official Website:** [https://www.ups.com.tr](https://www.ups.com.tr)

## Features

- International shipping
- Advanced tracking
- Reliable delivery

## Configuration Settings

To activate this integration, you need to provide the following information in the AVCI CMS admin panel:

| Field Name | Label | Type | Required | Description |
|------------|-------|------|----------|-------------|
| `customerNumber` | Customer Number | `text` | ✅ Yes | Ex: UPS Customer Number |
| `username` | Username | `text` | ✅ Yes | Ex: UPS Username |
| `password` | Password | `password` | ✅ Yes | Ex: UPS Password |
| `testMode` | Test Mode | `boolean` | ❌ No | - |

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.UPS` enum value. After completing the configuration via the admin panel, the system will automatically start using `UPS Kargo` services.
