# UPS Kargo 

UPS Kargo ile global teslimat çözümleri

**Resmi Web Sitesi:** [https://www.ups.com.tr](https://www.ups.com.tr)

## Özellikler

- Uluslararası gönderim
- Gelişmiş takip
- Güvenilir teslimat

## Konfigürasyon Ayarları

Bu entegrasyonu aktif etmek için AVCI CMS yönetim panelinde aşağıdaki bilgileri sağlamanız gerekmektedir:

| Alan Adı | Etiket | Tür | Zorunlu | Açıklama |
|----------|--------|-----|---------|----------|
| `customerNumber` | Müşteri Numarası | `text` | ✅ Evet | Örn: UPS Müşteri Numarası |
| `username` | Kullanıcı Adı | `text` | ✅ Evet | Örn: UPS Kullanıcı Adı |
| `password` | Şifre | `password` | ✅ Evet | Örn: UPS Şifresi |
| `testMode` | Test Modu | `boolean` | ❌ Hayır | - |

## Geliştirici & API Entegrasyonu

Bu entegrasyon AVCI CMS arka planında `IntegrationType.UPS` enum değeri ile yönetilmektedir. Yönetim paneli üzerinden yapılandırmayı tamamladıktan sonra sistem otomatik olarak `UPS Kargo` servislerini kullanmaya başlayacaktır.
