# DHL

DHL Express global shipping integration

**Official Website:** [https://www.dhl.com](https://www.dhl.com)

## Features

- International shipping
- Fast delivery
- International network

## Configuration Settings

To activate this integration, you need to provide the following information in the AVCI CMS admin panel:

| Field Name | Label | Type | Required | Description |
|------------|-------|------|----------|-------------|
| `siteId` | Site ID | `text` | ✅ Yes | Ex: DHL Site ID |
| `password` | Password | `password` | ✅ Yes | Ex: DHL Password |
| `testMode` | Test Mode | `boolean` | ❌ No | - |

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.DHL` enum value. After completing the configuration via the admin panel, the system will automatically start using `DHL Express` services.
