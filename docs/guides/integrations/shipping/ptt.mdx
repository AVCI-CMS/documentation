# PTT Kargo

PTT Kargo services integration

**Official Website:** [https://www.ptt.gov.tr](https://www.ptt.gov.tr)

## Features

- Economical shipping
- Wide branch network
- Postal services

## Configuration Settings

To activate this integration, you need to provide the following information in the AVCI CMS admin panel:

| Field Name | Label | Type | Required | Description |
|------------|-------|------|----------|-------------|
| `customerNumber` | Customer Number | `text` | ✅ Yes | Ex: PTT Customer Number |
| `password` | Password | `password` | ✅ Yes | Ex: PTT Password |
| `testMode` | Test Mode | `boolean` | ❌ No | - |

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.PTT` enum value. After completing the configuration via the admin panel, the system will automatically start using `PTT Kargo` services.
