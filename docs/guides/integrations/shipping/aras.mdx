# Aras Kargo

Fast and reliable delivery solutions with Aras Kargo infrastructure

**Official Website:** [https://www.araskargo.com.tr](https://www.araskargo.com.tr)

## Features

- Automatic barcode generation
- Real-time status tracking
- Regional pricing
- Return process management

## Configuration Settings

To activate this integration, you need to provide the following information in the AVCI CMS admin panel:

| Field Name | Label | Type | Required | Description |
|------------|-------|------|----------|-------------|
| `username` | Username | `text` | ✅ Yes | Ex: Aras Kargo Username |
| `password` | Password | `password` | ✅ Yes | Ex: Aras Kargo Password |
| `customerCode` | Customer Code | `text` | ❌ No | Ex: Aras Kargo Customer Code |
| `testMode` | Test Mode | `boolean` | ❌ No | - |

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.ARAS` enum value. After completing the configuration via the admin panel, the system will automatically start using `Aras Kargo` services.
