# Sürat Kargo

Sürat Kargo API integration

**Official Website:** [https://www.suratkargo.com.tr](https://www.suratkargo.com.tr)

## Features

- Cargo tracking
- Automatic shipping
- Web service

## Configuration Settings

To activate this integration, you need to provide the following information in the AVCI CMS admin panel:

| Field Name | Label | Type | Required | Description |
|------------|-------|------|----------|-------------|
| `password` | Password | `password` | ✅ Yes | Ex: Sürat Kargo Password |
| `testMode` | Test Mode | `boolean` | ❌ No | - |

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.SURAT` enum value. After completing the configuration via the admin panel, the system will automatically start using `Sürat Kargo` services.
