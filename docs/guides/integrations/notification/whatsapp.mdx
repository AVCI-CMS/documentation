# WhatsApp Notifications

Send WhatsApp messages to your customers via Meta Business API

**Official Website:** [https://business.whatsapp.com](https://business.whatsapp.com)

## Features

- Meta Cloud API support
- Approved template messages
- High delivery rate
- Interactive button support

## Configuration Settings

To activate this integration, you need to provide the following information in the AVCI CMS admin panel:

| Field Name | Label | Type | Required | Description |
|------------|-------|------|----------|-------------|
| `accessToken` | Permanent Access Token | `password` | ✅ Yes | Ex: EAAG... |
| `phoneNumberId` | Phone Number ID | `text` | ✅ Yes | Ex: 123456789012345 |
| `wabaId` | WhatsApp Business Account ID | `text` | ✅ Yes | Ex: 123456789012345 |
| `defaultLanguage` | Default Language Code | `text` | ❌ No | Ex: en |
| `testNumber` | Test Phone Number | `text` | ❌ No | Ex: 905551234567 |

## Setup Steps

### 1. Create a Meta App
Create a Business App from developers.facebook.com and add the WhatsApp product.

### 2. Get API Information
Copy the Phone Number ID and Access Token information from the WhatsApp -> Getting Started section.

### 3. Save Your Phone Number
Add your own phone number as a test recipient and enter the API information here.

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.WHATSAPP` enum value. After completing the configuration via the admin panel, the system will automatically start using `WhatsApp Business` services.
