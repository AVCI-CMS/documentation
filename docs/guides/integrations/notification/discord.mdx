# Discord Bildirimleri

Discord sunucularına webhook ile bildirimler gönderin

**Resmi Web Sitesi:** [https://discord.com](https://discord.com)

## Özellikler

- Webhook entegrasyonu
- Embed mesajları
- Kanal bildirimleri
- Rich formatting

## Konfigürasyon Ayarları

Bu entegrasyonu aktif etmek için AVCI CMS yönetim panelinde aşağıdaki bilgileri sağlamanız gerekmektedir:

| Alan Adı | Etiket | Tür | Zorunlu | Açıklama |
|----------|--------|-----|---------|----------|
| `webhookUrl` | Webhook URL | `url` | ✅ Evet | Örn: https://discord.com/api/webhooks/... |
| `channel` | Kanal | `text` | ✅ Evet | Örn: #genel |
| `username` | Bot Adı | `text` | ❌ Hayır | Örn: AVCI CMS Bot |

## Kurulum Adımları

### 1. Kanal Ayarlarına Gidin
Bildirim almak istediğiniz kanalın ayarlarına (dişli çark) girin.

### 2. Entegrasyonlar ve Webhook
Integrations -> Webhooks kısmına gidin ve 'New Webhook' butonuna tıklayın.

### 3. URL'yi Kopyalayın
Webhook URL'sini kopyalayıp aşağıdaki yapılandırma alanına yapıştırın.

## Geliştirici & API Entegrasyonu

Bu entegrasyon AVCI CMS arka planında `IntegrationType.DISCORD` enum değeri ile yönetilmektedir. Yönetim paneli üzerinden yapılandırmayı tamamladıktan sonra sistem otomatik olarak `Discord` servislerini kullanmaya başlayacaktır.
