# Verimor SMS

Send SMS with Turkey's reliable SMS gateway service

**Official Website:** [https://www.verimor.com.tr](https://www.verimor.com.tr)

## Features

- Bulk SMS delivery
- OTP SMS
- Marketing SMS
- SMS reporting
- API integration
- Balance inquiry

## Configuration Settings

To activate this integration, you need to provide the following information in the AVCI CMS admin panel:

| Field Name | Label | Type | Required | Description |
|------------|-------|------|----------|-------------|
| `username` | Username | `text` | ✅ Yes | Ex: verimor_username |
| `password` | Password | `password` | ✅ Yes | Ex: verimor_password |
| `defaultSender` | Default Sender | `text` | ❌ No | Ex: AVCI CMS |

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.VERIMOR` enum value. After completing the configuration via the admin panel, the system will automatically start using `Verimor` services.
