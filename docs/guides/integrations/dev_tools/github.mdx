# GitHub

Access your private repositories via GitHub and pull content/code to CMS

**Official Website:** [https://github.com](https://github.com)

## Features

- Private repo access
- OAuth2 security
- Markdown/Code content import

## Developer & API Integration

This integration is managed in the AVCI CMS backend with the `IntegrationType.GITHUB` enum value. After completing the configuration via the admin panel, the system will automatically start using `GitHub` services.
