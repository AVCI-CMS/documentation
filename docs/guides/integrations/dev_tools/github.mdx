# GitHub 

GitHub üzerinden private repolarınıza erişin ve CMS'e içerik/kod çekin

**Resmi Web Sitesi:** [https://github.com](https://github.com)

## Özellikler

- Private repo erişimi
- OAuth2 güvenliği
- Markdown/Kod içeriği aktarımı

## Geliştirici & API Entegrasyonu

Bu entegrasyon AVCI CMS arka planında `IntegrationType.GITHUB` enum değeri ile yönetilmektedir. Yönetim paneli üzerinden yapılandırmayı tamamladıktan sonra sistem otomatik olarak `GitHub` servislerini kullanmaya başlayacaktır.
