# Meta (Facebook) Pixel

Track conversions for your Facebook and Instagram ad campaigns.

## Setup
1. From Meta Business Manager, copy your **Pixel ID**.
2. To improve tracking accuracy against ad-blockers, generate a **Conversions API Access Token**.
3. In AVCI CMS, go to **Integrations > Analytics**.
4. Enter both your Pixel ID and the Conversions API Token. The CMS will automatically track events using both client-side and server-side methods.
