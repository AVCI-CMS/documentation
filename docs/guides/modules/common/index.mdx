# Common Modules

The **Common** group contains core building blocks that do not exist as independent root menu items in AVCI CMS but serve as tabs or sub-modules under various other modules such as **Catalog (Products)**, **Contents (Blog)**, or **Relations (Merchants)**.

For example, the "Categories" or "Comments" logic operates with the exact same infrastructure and interface within products, blog posts, or news. In this guide, you can learn how these common modules are managed.
