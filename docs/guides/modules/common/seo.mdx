# SEO Settings

To help you represent your content (products, categories, pages, blogs, news, etc.) in the best possible way on search engines (Google, Bing, etc.), AVCI CMS includes a common **SEO** tab or section within almost every main module.

When you enter the details of a content item (e.g., a product), you will generally see the "SEO" heading in the top tabs or the side menu.

## SEO Fields

You can optimize the following fields to increase search engine visibility:

- **SEO Title:**
  This is the main title that will appear on Search Engine Results Pages (SERP) and in the browser tab. It is usually written to be longer than the normal product/content name and includes search-oriented keywords. If left blank, the system automatically uses the content's standard title.

- **SEO Description:**
  This is the summary text (Meta Description) that appears just below the title in search results. It is recommended to write a compelling 150-160 character summary that will persuade the user to click.

- **URL (Slug):**
  This is the path of the content that appears in the browser address bar (e.g., `example-product-name`).
  - It should not contain special characters, spaces, or non-English characters.
  - If you leave it blank, the system will automatically generate an SEO-friendly slug based on the main title of the content.
  - Keep in mind that if you change an existing slug, you may need to set up a manual 301 redirect to prevent users visiting the old URL from encountering a 404 error.

## Tips
> **Auto-Fill:** When adding new content, if you only fill in the "Title" field, the SEO Title and Slug are automatically generated by the system. You only need to edit these fields if you want to explicitly override the defaults.
