# Coupons

**Menu Path:** `Catalog > Coupons`

The **Coupons** module is used to create specific, trackable alphanumeric codes that users can enter at checkout to receive a discount. Unlike Campaigns, which apply automatically, coupons require user action and are excellent tools for targeted marketing, affiliate tracking, and customer retention.

## Creating a Discount Coupon

To generate a new coupon code:

1. Navigate to **Catalog > Coupons**.
2. Click **"Add New"**.

### 1. Basic Info & Discount Value
- **Code:** The actual text the user will type (e.g., `WELCOME20`, `SUMMERSALE`). This field automatically converts text to uppercase.
- **Discount Type:** Choose how the discount is applied:
  - *PERCENTAGE:* Reduces the cart total by a percentage (e.g., 20%).
  - *FIXED:* Reduces the cart total by a flat currency amount (e.g., $15).
  - *FREE_SHIPPING:* Waives the shipping fees.
- **Discount Value:** Enter the numeric value corresponding to the chosen type.
- **Active:** Toggle the coupon on or off instantly.
- **Auto Apply:** If toggled, the system will attempt to automatically apply this coupon to eligible carts without the user needing to type it.

### 2. Validity & Limits
- **Valid From & Valid Until:** Set the precise date and time window when the coupon is active.
- **Min Order Amount:** The cart subtotal must reach this value before the coupon is valid.
- **Max Discount Amount:** Useful for percentage discounts. (e.g., "20% off, up to a maximum of $50").
- **Max Usage:** The total number of times this coupon can be used across the entire platform (e.g., "First 100 customers only").
- **Usage Per Customer:** How many times a single registered user can use this specific code (usually set to `1`).

### 3. Usage Terms
- **Terms List:** Add line-by-line legal or explanatory terms associated with this coupon (e.g., "Cannot be combined with other offers", "Valid only on full-priced items").

3. Once configured, click **"Save"** to activate the coupon.
