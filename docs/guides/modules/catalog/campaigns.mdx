# Campaigns

**Menu Path:** `Catalog > Campaigns`

The **Campaigns** module allows you to configure global discount rules that apply automatically during checkout. Unlike static product discounts, Campaigns are dynamic and time-sensitive.

## Creating a Campaign

To set up a new promotional campaign:

1. Go to **Catalog > Campaigns**.
2. Click **"Add New"**.

### 1. Basic Info & Dates
- **Title:** The internal or public name of the campaign (e.g., "Black Friday 2026").
- **Status:** *Draft*, *Active*, *Inactive*, *Scheduled*, or *Completed*.
- **Description:** A short explanation of the campaign rules.
- **Start Date & End Date:** Set the exact window during which this campaign is valid.

### 2. Discount Value
- **Discount Percentage (%):** e.g., 20 for a 20% discount.
- **Discount Amount ($):** A fixed currency deduction (e.g., $50 off). 
*(Note: Use either Percentage or Amount based on your strategy).*

### 3. Conditions
- **Conditions List:** Define the specific criteria required for the campaign to trigger. This can include specific product categories, minimum cart values, or user groups. 

3. Click **"Save"** to schedule or activate the campaign.
