# Events

**Menu Path:** `Catalog > Events`

The **Events** module is a comprehensive ticketing and event management system. It allows you to create everything from a one-off webinar to a recurring weekly workshop, or a massive physical concert spanning multiple days.

## Creating an Event

To create a new event:

1. Navigate to **Catalog > Events**.
2. Click the **"Add New"** button.
3. The event builder is divided into multiple logical cards to handle the complexity of event management:

### 1. General Info
- **Event Name:** The headline of your event.
- **Status:** *Draft*, *Published*, *Hidden*, or *Deleted*.
- **Category:** Group your events (e.g., "Concerts", "Workshops").
- **Event Type:** Choose between *Physical* (requires a Venue) or *Digital* (requires an online link).
- **Date & Time:** Set the primary Start Date and End Date.
- **Event Mode (Critical!):** 
  - *Single:* A one-time event.
  - *Serial:* A multi-part event where you manually define individual sessions.
  - *Recurring:* Automatically generates sessions based on a rule (e.g., "Every Friday for 10 weeks").
- **Merchant/Organizer:** Link the event to a specific organizer or merchant account.

### 2. Media
- **Featured Image:** The main poster or thumbnail for the event.
- **Background/Banner Image:** High-resolution images used for the event detail page header.
- **Gallery:** Additional photos from past events or promotional graphics.

### 3. Venue (Location)
- If the event is *Physical*, you must link it to a specific **Place** (managed in the [Relations > Places](../relations/places.mdx) module). This automatically pulls the map coordinates, address, and seating arrangements for the event.

### 4. Sessions (For Serial/Recurring Events)
- If you selected Serial or Recurring mode, this card appears. You can define specific sub-sessions, label them (e.g., "Day 1: Keynote", "Day 2: Workshops"), and set their exact start and end times.

### 5. Pricing & Capacity
- **Capacity limits:** Define the Minimum (for the event to happen), Maximum (total tickets available), and currently Reserved tickets.
- **Base Price:** Set the ticket price, currency, and VAT rules.

### 6. Description & Content
- **Description:** Use the rich text editor to provide a full event itinerary, speaker bios, and marketing content.
- **Banner Generator:** A built-in tool to dynamically generate social media sharing banners for this specific event.

### 7. Notes & Rules
- **Notes:** Important information the attendee must know before purchasing (e.g., "Please arrive 30 minutes early").
- **Rules:** Strict policies (e.g., "No age under 18 allowed", "No refunds 48 hours prior to the event").

4. Review your settings and click **"Save"** to finalize the event creation.
