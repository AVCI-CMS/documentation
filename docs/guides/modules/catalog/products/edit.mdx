# Edit Product & Advanced Configuration

**Menu Path:** `Catalog > Products > Edit`

Once a product is created, clicking the **"Edit"** button from the Products List opens a comprehensive configuration interface organized by tabs. This is where you unlock the full potential of your catalog management.

## 1. Pricing Configuration Tab
Set how much the product costs and handle tax implications.

- **Base Price:** The standard retail price of the product.
- **Discounted Price (Sale Price):** If you are running a specific promotion without using the Campaigns module, enter the lower price here. The frontend will display the base price crossed out.
- **Tax Rate:** Select the applicable VAT or sales tax percentage. Determine if the entered base price is *Tax Inclusive* or *Tax Exclusive* (based on your Workspace settings).
- **Cost Price:** (Internal use) Track how much the item costs you to calculate profit margins in reports.

## 2. Attributes Tab
Attributes define the technical specifications and features of a product. They are often used for frontend filtering (e.g., filtering laptops by RAM size).

- **Adding an Attribute:** Click "Add Attribute". Select the Attribute Key (e.g., `Material`) and define the Value (e.g., `100% Cotton`).
- **Visibility:** You can choose whether an attribute is displayed publicly on the product page or kept internal.

## 3. Variants (Options) Tab
If your product comes in multiple options (like a t-shirt available in S, M, L and Red, Blue), you must define variants.

1. **Define Option Types:** First, create the Option Types (e.g., "Size", "Color").
2. **Assign Values:** For "Size", add values "S", "M", "L". For "Color", add "Red", "Blue".
3. **Generate Variants:** The system will automatically generate a matrix of sub-products (e.g., "Red-S", "Blue-M").
4. **Variant Specifics:** For each generated variant row, you can define a unique SKU, independent stock quantity, specific pricing (e.g., XL costs $5 more), and assign a specific variant image (the image switches when the customer selects "Red").

## 4. SEO & Meta Tags Tab
Optimize how search engines index your product page.

- **Slug (URL Path):** Automatically generated from the product name, but can be manually overridden to be shorter or keyword-rich (e.g., `/mens-cotton-tshirt`).
- **Meta Title:** The title displayed on the browser tab and Google search results (Recommended: 50-60 characters).
- **Meta Description:** A compelling summary that encourages users to click the link in search results (Recommended: 150-160 characters).

## 5. Related Products & Cross-Sells
Boost your average order value by recommending other items.

- **Upsells:** Recommend more expensive alternatives to the current product.
- **Cross-sells:** Recommend complementary items (e.g., suggesting a phone case on a smartphone product page). Search and select products directly from the catalog to link them.

> **Save Frequently:** Always ensure you click **"Save Changes"** at the bottom of the screen before navigating away from the edit interface.
