# Events List

**Menu Path:** `Catalog > Events`

The **Events** module is a specialized catalog structure designed for time-bound experiences. Whether you are hosting a physical concert, a multi-day conference, or an online webinar, this module handles scheduling, ticketing, and attendee management.

## Event Management Dashboard

When you navigate to `Catalog > Events`, you are presented with a chronological listing of all your events.

### Filtering the Event View
Given the time-sensitive nature of events, filtering is crucial:
- **Date Range Filters:** Quickly view "Upcoming Events", "Past Events", or events occurring "This Month".
- **Status Filters:** Differentiate between `Published` events (currently selling tickets), `Drafts`, and `Cancelled` events.
- **Type Filters:** Filter by Online (Virtual) vs. Physical (Venue-based) events.

### At-a-Glance Metrics
The data table provides instant insights without needing to click into an event:
- **Sales Progress:** See a progress bar or ratio of Tickets Sold vs. Total Capacity.
- **Date & Time:** Instantly verify the start and end dates.

## Creating a New Event
To begin planning a new gathering, click the **"Create"** button in the top right corner. The creation process involves defining the core concept, time, and location, which is detailed in the [Create Event](./create.mdx) section.
