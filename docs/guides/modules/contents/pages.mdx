# Pages

**Menu Path:** `Contents > Pages`

The **Pages** module is the section where you create permanent and structural contents of your website such as "About Us", "Contact", "Terms of Use", "Privacy Policy".

Unlike blogs and news, pages are generally contents that do not have a specific chronological order and form the basic structure of the site.

## How to Create a New Page?

1. Follow the **Contents > Pages** step from the left menu.
2. Click the **"Create"** button in the upper right of the opened Pages list.
3. Fill in the following sections on the page detail screen:
   - **Title:** The visible name of the page (e.g., Who Are We?).
   - **Slug:** The URL structure of the page (e.g., `/who-are-we`). It is usually generated automatically from the title, but you can edit it as you wish.
   - **Content:** The rich text editor (or block design tool if you use one) where you will create the page body.
4. **SEO and Visibility:** Enter the Meta Title and Description parts to determine how the page will appear in search engines.
5. Make its Status active and press the **Save** button to publish the page.

## Using Page Templates

If you have custom page designs defined in your system (e.g., Template with Contact Form, Full Screen Image Template), you can select the relevant design from the **"Template"** option in the right menu while creating or editing the page. This ensures that your content is rendered with the correct design on the frontend.
