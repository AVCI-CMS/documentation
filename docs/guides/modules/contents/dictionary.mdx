# Dictionary

**Menu Path:** `Contents > Dictionary`

The **Dictionary** module allows you to create a comprehensive glossary of terms relevant to your industry. This is an exceptionally powerful tool for Search Engine Optimization (SEO). It helps you rank for specific industry keywords by providing valuable, easily digestible definitions to your users. 

Furthermore, the Dictionary can act as a centralized data source for the [Attributes](../common/attributes.mdx) module (e.g., creating a "Fabric Types" dictionary to be used as product variations).

## Adding a New Dictionary Term

To add a new word or phrase to your glossary:

1. Navigate to **Contents > Dictionary**.
2. Click the **"Add New"** button to open the term creation form.
3. Fill in the required fields across the different sections.

### 1. Basic Info
- **Name:** The actual word or term you are defining (e.g., "Headless CMS").
- **Status:** Choose whether the term is *Published* (live on the site), *Draft*, *Hidden*, or *Deleted*.
- **Category:** Select the appropriate category (from the [Common Categories](../common/categories.mdx) module) to group related terms together.
- **Type:** The grammatical type or category of the term (e.g., Noun, Verb, Adjective, or an industry-specific classification).
- **Slug:** The URL-friendly identifier (e.g., `headless-cms`). If left blank, it is automatically generated from the Name.
- **Phonetic:** The phonetic transcription of how the word is pronounced (e.g., `/ˈhedləs/`).

### 2. Media
- **Featured Image:** A primary image that visually represents the term. This is often used as the thumbnail in glossary lists.
- **Gallery:** Additional images or diagrams that help explain the term.

### 3. Definition (Rich Text)
- Provide the core explanation of the term. You can use the Rich Text editor to format the text, add lists, embed links to other terms, and insert inline images to make the definition as clear as possible.

### 4. Etymology
- A plain text area where you can explain the origin of the word or how the term came into use within your industry.

### 5. Example
- Provide concrete sentences or use cases showing how the term is applied in real life. This significantly helps users understand abstract concepts.

4. Once all relevant fields are filled, click the **"Save"** button at the top right to create the dictionary term.
