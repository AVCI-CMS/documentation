# Notifications

**Menu Path:** `Communication > Notifications`

While the Newsletter module is for long-form email marketing, the **Notifications** module is designed for short, urgent, and highly targeted alerts.

## Sending a Notification

1. Navigate to **Communication > Notifications**.
2. Click **"Add New"**.

### 1. Message Content
- **Title:** The bold headline of the notification.
- **Body:** A brief message (e.g., "Your order has been shipped!", "Flash Sale ends in 2 hours!").
- **Target URL (Action):** Where should the user be redirected when they click the notification? (e.g., A specific product page or their order tracking screen).

### 2. Notification Channels
You can choose to push the message through one or multiple channels:
- **In-App (Bell Icon):** Sends the message to the notification tray within your website or mobile app.
- **Push Notification:** Sends an OS-level push alert to the user's mobile device or browser (requires the user to have opted-in).
- **SMS:** Sends a text message. *(Note: This incurs per-message costs depending on your SMS provider).*

### 3. Targeting
- **Global:** Send to all active users.
- **Targeted:** Send to a specific User ID, a specific group, or filter by behavior.

3. Click **"Send"** to dispatch the notification instantly.
