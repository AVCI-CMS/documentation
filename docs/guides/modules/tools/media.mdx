# Media

**Menu Path:** `Tools > Media`

The **Media** module is the centralized repository for every digital file uploaded to the platform. Whether you upload a product thumbnail, a blog banner, or a downloadable PDF manual, it physically resides here.

## Managing Media

1. Go to **Tools > Media**.
2. You will see a grid or list of all uploaded files.

### 1. Uploading Files
- You can upload files directly here to be used later in Rich Text editors, sliders, or page builders.
- The system automatically handles image compression and generates multiple sizes (e.g., small thumbnail, large banner) for performance optimization, ensuring your website loads quickly.

### 2. Organizing
- Use folders to keep your media structured (e.g., `2026_Campaigns`, `Product_Manuals_PDF`).

### 3. Cleaning Up
- If you delete a product or a blog post, its images might still remain in the Media library to prevent breaking other pages that might accidentally reference them.
- You can safely use the Media module to delete old, unused files to save server storage space. *Warning: Deleting a file here will instantly break the image on any frontend page that is still using it.*
