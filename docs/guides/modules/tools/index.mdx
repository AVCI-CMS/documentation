# Tools

The **Tools** module group contains utility features that support the core operations of the CMS. These are auxiliary systems that help you manage the visual presentation of your site and handle external integrations like messaging and physical interactions.

### Modules in this Group

- **Slider:** Create and manage dynamic image and video sliders (carousels) for your frontend homepage and landing pages.
- **Media:** The central repository for all files, images, PDFs, and videos uploaded to the platform.
- **Mail:** Configure email templates, SMTP settings, and view logs of transactional emails sent by the system.
- **SMS:** Manage SMS gateway configurations and broadcast urgent text messages to your users.
- **NFC:** Register and manage physical Near Field Communication (NFC) tags for events or smart products.
- **QRCode:** Generate and track dynamic QR codes for physical marketing materials.
