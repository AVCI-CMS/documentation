# Slider

**Menu Path:** `Tools > Slider`

The **Slider** module allows you to build dynamic, rotating image and video carousels. These are most commonly used at the very top of your homepage ("Hero Sliders") to highlight current campaigns, new products, or upcoming events.

## Creating a Slider

1. Navigate to **Tools > Slider**.
2. Click **"Add New"** to create a new slider group (e.g., "Homepage Hero Slider").

### Managing Slides
Inside a slider group, you can add multiple individual slides:
- **Image/Video:** Upload the desktop and mobile versions of the banner. 
- **Titles & Text:** Add an overlay heading and subheading.
- **Call to Action (CTA):** Configure the button text (e.g., "Shop Now") and the target URL.
- **Order:** Drag and drop the slides to change the sequence they appear in.

## Using Sliders on the Frontend
Once published, these sliders are dynamically fetched by the frontend application based on their unique identifier (Slug). This allows the marketing team to change homepage banners at any time without requiring a developer to write code.
