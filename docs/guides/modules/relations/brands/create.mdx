# Create Brand

**Menu Path:** `Relations > Brands > Create`

To add a new brand to your platform's catalog, click the **"Create"** button on the Brands list page.

## Defining a Brand

1. **Brand Name:** Enter the official, properly capitalized name of the brand (e.g., "Sony"). This exact text will appear in the frontend filters.
2. **Brand Logo:** Upload a high-quality, transparent PNG of the brand's logo. This is highly recommended because many themes display a "Shop by Brand" carousel on the homepage using these images.
3. **Brand Description:** Write a short biography or history of the brand.
   - *SEO Benefit:* If your frontend generates dedicated brand pages (e.g., `yourstore.com/brand/sony`), this description serves as the on-page content, greatly boosting your search engine ranking for that brand's keywords.
4. **Status:** Ensure the status is set to **Active**.
5. Click **"Save"**. The brand is immediately available for selection when creating or editing a [Product](../../catalog/products/edit.mdx).
