# Brands List

**Menu Path:** `Relations > Brands`

The **Brands** module manages the central directory of all manufacturer and brand names associated with the products in your catalog. A clean, structured brand directory is vital for frontend filtering, search accuracy, and SEO.

## Why Use a Brand Directory?
Instead of letting merchants type "Nike", "nike", or "NIKE" manually into a text field when creating a product, they must select from this pre-defined list. This ensures data consistency across the entire platform.

## Navigating the Brand Directory

The data table under `Relations > Brands` provides an overview of your active brands.

- **Brand Name:** The official, standardized name.
- **Product Count:** See exactly how many active products in your catalog are currently linked to this brand.
- **Status:** Toggle brands between Active and Inactive. If a brand is marked as Inactive, it will disappear from frontend filter sidebars (though existing products may still be accessible via direct link).

## Managing the List
You can use the bulk action checkboxes to delete or deactivate multiple brands at once if you are cleaning up the catalog. For adding a new brand, click the **"Create"** button.
