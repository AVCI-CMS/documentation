# Media Library

**Menu Path:** `Tools > Media`

The **Media** module is the central repository for all digital assets (images, documents, PDFs, and videos) uploaded across the entire AVCI CMS platform.

## Why Use the Media Library?
Instead of uploading the same "Free Shipping" icon multiple times on different product pages, you upload it once to the Media Library and reuse it. This saves server space and improves site performance.

## Organizing Your Assets

When you open the Media module, you are presented with a file explorer interface.

### 1. Folder Management
To keep your assets organized (especially if you have thousands of products):
- Click **"New Folder"**.
- Create logical structures like `/products/shoes/2026` or `/blog-images/august`.
- You can drag and drop existing files into these folders.

### 2. Image Optimization
The CMS automatically processes images to ensure fast loading times on the frontend.
- When you click on an image, you can see its dimensions (e.g., 800x800px) and file size.
- If a file is unnecessarily large (e.g., a 5MB raw photo), you can use the built-in crop and compress tools to reduce it before assigning it to a product.

### 3. Deleting Unused Media
Over time, your server can fill up with old promotional banners.
- Look for files that have `0` in the "Attached To" column. This means the image is not currently used on any product, blog, or page.
- You can safely delete these files to free up storage space.
