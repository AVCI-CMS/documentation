# Brand Settings

**Route:** `(brand) > /brand/settings`

The Settings module allows you to configure operational parameters and permissions for your brand across the platform.

## Configuration Options

- **Catalog Exclusivity:** Toggle whether *any* merchant can list your products, or if only explicitly authorized merchants (whitelisted) are permitted to sell your brand.
- **Notification Preferences:** Subscribe to automated reports (e.g., weekly sales summaries across all merchants selling your brand) or compliance alerts.

:::info Feature Flagging
The "Catalog Exclusivity" feature is only visible if the Workspace Admin has enabled strict brand gating for the platform. If you do not see this option, the platform operates on an open-listing model.
:::
