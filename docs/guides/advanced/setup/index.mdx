# Welcome to the Onboarding Wizard

AVCI CMS uses a mandatory **Onboarding Wizard** for all new entities. Before you can access the dashboard, view orders, or add products, you must complete the setup process tailored to your specific account type.

Because the legal and operational requirements differ vastly between a top-level Workspace and an individual Marketplace Seller, the system routes you through different forms.
