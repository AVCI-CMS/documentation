# 4. Documents

The required file uploads change dynamically based on the Corporation Type selected in Step 2.

### Standard Requirements
- **Identity Document:** Photo ID of the applicant.
- **Signature Circular:** Proof that the applicant has the authority to sign for the company.
- **Marketplace Agreement:** You must download the generated PDF, sign it, and upload the signed copy.

### Conditional Requirements
- **Tax Certificate:** Required unless the user checks the "Tax Exemption" box.
- **Trade Registry Gazette:** Required *only* for Limited and Joint Stock companies.
