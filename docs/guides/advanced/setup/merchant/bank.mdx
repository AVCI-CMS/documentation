# 3. Bank Info

Required for the marketplace to process payouts (commissions transferred to the seller).

- **Account Holder:** The name on the bank account *must* match the Corporate Name (or Applicant Name for personal companies). Payouts to third-party accounts are blocked by default.
- **IBAN Validation:** The IBAN must start with the correct country code (e.g., `TR`) and pass mathematical checksum validations.
