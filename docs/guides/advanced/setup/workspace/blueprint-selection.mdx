# 1. Blueprint Selection

The very first step in setting up a new Workspace is choosing your architectural **Blueprint**.

- **Plan Selection:** Choose between standard SaaS plans or Enterprise dedicated clusters.
- **Template:** Select the core use-case (e.g., "B2C E-commerce", "B2B Wholesale", "Marketplace"). This automatically pre-configures hundreds of internal settings.
- **Initialization:** Once selected, the system provisions your isolated database schema in the background.
