# 6. Completion & API Keys

Once all steps are verified, the wizard finalizes the setup.

## Critical Security Warning

On the final success screen, you will be presented with your master **Client ID** and **API Key**.

- **Save it immediately:** The API Key is shown **only once**. If you lose it, you will need to regenerate a new one from the Advanced Settings, which will break any existing frontend connections.
- **Next Steps:** Copy these credentials into the `.env` file of your Headless Next.js frontend.
