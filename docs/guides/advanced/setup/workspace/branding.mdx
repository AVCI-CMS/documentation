# 3. Branding

Set up the visual identity of your new CMS instance.

- **Workspace Name:** Enter the official name of your platform (e.g., "ACME Global").
- **Logos:** Upload your primary logo and favicon. These will be used both in the admin dashboard and injected into transactional emails.
- **Primary Colors:** Select the hex codes that match your corporate identity.
